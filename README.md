# React Web App CI/CD Pipeline with GitLab and AWS
This project demonstrates a complete CI/CD pipeline for deploying a React application using GitLab CI/CD and AWS services.

## Tech Stack
Framework: React \n
CI/CD Orchestration: GitLab CI/CD \n
Containerization: Docker (Uses nginx:1.29.3-alpine to serve the React build).\n
Static website hosting for staging: AWS S3 \n
Production deployment: AWS Elastic Beanstalk (EB) \n
Infrastructure and access control: AWS EC2 & IAM \n
