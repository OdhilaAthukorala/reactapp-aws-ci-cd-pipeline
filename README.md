# React Web App CI/CD Pipeline with GitLab and AWS
This project demonstrates a complete CI/CD pipeline for deploying a React application using GitLab CI/CD and AWS services.

## Tech Stack
Framework: React
CI/CD Orchestration: GitLab CI/CD
Containerization: Docker (Uses nginx:1.29.3-alpine to serve the React build).
Static website hosting for staging: AWS S3
Production deployment: AWS Elastic Beanstalk (EB)
Infrastructure and access control: AWS EC2 & IAM
